<StepByStepSolution
  client:load
  hint="Jmenovatel zlomku nesmí být nikdy roven nule."
  steps={[

<StepByStepSolution

## Příklad 4: Odmocnina ve jmenovateli

> 📝 **Zadání:** Urči definiční obor funkce $f(x) = \dfrac{1}{\sqrt{x - 3}}$.

<StepByStepSolution
  client:load
  hint="Musí platit dvě podmínky: pod odmocninou ≥ 0 a zároveň jmenovatel ≠ 0."
  steps={[
    "Předpis funkce: <strong>f(x) = 1/√(x − 3)</strong>.",
    "Pod odmocninou nesmí být záporné číslo: <strong>x − 3 ≥ 0</strong>.",
    "Zároveň jmenovatel nesmí být nula: √(x − 3) ≠ 0 ⇒ x − 3 ≠ 0 ⇒ <strong>x ≠ 3</strong>.",
    "Spojíme podmínky: x ≥ 3 a zároveň x ≠ 3 ⇒ <strong>x > 3</strong>.",
    "<strong>Odpověď:</strong> Definiční obor je <strong>D(f) = (3, +∞)</strong>."
  ]}
/>
